### 可以调整的内容

#### 1. 更新频率
编辑 `.github/workflows/auto-update.yml`：
```yaml
# 每天 8:00
- cron: '0 0 * * *'

# 每 6 小时
- cron: '0 */6 * * *'

# 每小时
- cron: '0 * * * *'
```

#### 2. 抓取项目数量
编辑 `scripts/fetch_github.py`：
```python
for article in articles[:10]:  # 改为 [:20] 抓取 20 个
```

#### 3. 网站样式
编辑 `index.html` 中的 CSS 部分

#### 4. 添加新数据源
